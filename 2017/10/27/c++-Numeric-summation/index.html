数列求和

发表于 2017-10-27 | 分类于 C++

问题描述

i=0 to n-1 : ∑x^i mod p

范围：1<=x<=10^6 1<=n<=10^12 1<=p<=10^6

例如： x = 4 n = 2 p = 3 结果为：1 + 4 = 2 (mod 3)

算法思路

大数，不能直接按照公式算，需要化简

(x^0 + x^1 + …… + x^n-1) mod p =
(x^0 mod p + x^1 mod p + …… + x^n-1 mod p) mod p =
((x mod p)^0 mod p + (x mod p)^1 mod p + …… + (x mod p)^n-1 mod p) mod p

发现 (x mod p)^n-1 mod p 求解任然太大

有 (a b) mod c = (a mod c b mod c) mod c

故 (x mod p)^n-1 mod p = ((x mod p)^n-2 mod p) * ((x mod p) mod p) mod p = ……
一次迭代下去即可。

同样 在最后 求和的时候
有 (a + b)mod c = (a mod c + b mod c) mod c
